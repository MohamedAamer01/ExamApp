ExamApp

Application web de création et de passage d’examens en ligne développée avec Java EE.

📌 Description

ExamApp est une plateforme permettant aux administrateurs de créer et gérer des examens en ligne, tandis que les étudiants peuvent passer les examens directement depuis leur navigateur dans un environnement chronométré.

Le projet suit une architecture MVC et utilise les technologies Java EE telles que JSP, Servlets, JPA/Hibernate et MySQL.

🚀 Fonctionnalités
👨‍🏫 Administrateur
Authentification sécurisée
Création et gestion des examens
Ajout et suppression de questions
Gestion des questions QCM et réponses courtes
Importation des étudiants via fichier Excel
Consultation des résultats des étudiants
👨‍🎓 Étudiant
Authentification
Consultation des examens disponibles
Passage des examens avec chronomètre
Affichage immédiat des résultats

🗄️ Base de données

Le projet utilise MySQL pour gérer :

les utilisateurs,
les examens,
les questions,
les réponses,
les résultats.
▶️ Exécution du projet
Cloner le dépôt :
git clone <repo-url>
Importer le projet dans Eclipse IDE.
Configurer la base de données MySQL.
Déployer le projet sur Apache Tomcat.
Lancer l’application depuis :
http://localhost:8080/
